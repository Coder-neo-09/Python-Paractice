## ğŸ“Œ 1. Instance Attributes

* Belong to **specific objects**
* Defined inside `__init__` using `self`

```python

    def __init__(self, brand, color):
        self.brand = brand
        self.color = color

print(my_car.color)  # Red
```

## ğŸ“Œ 2. Class Attributes

* Shared by **all objects** of the class
* Defined directly inside the class, outside `__init__`

```python
class Car:
    wheels = 4  # Class attribute

car1 = Car()
car2 = Car()
print(car1.wheels)  # 4
print(car2.wheels)  # 4
```

## ğŸ“Œ 4. Class Methods

* Defined with `@classmethod` decorator
* First parameter is `cls` â†’ refers to the class itself

```python

    wheels = 4

    @classmethod
    def info(cls):
        print(f"All cars have {cls.wheels} wheels.")

```
