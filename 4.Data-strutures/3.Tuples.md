## ğŸ”„ Tuple Operations

### Accessing Items (Indexing)

```python
fruits = ("apple", "banana", "cherry")
print(fruits[0])     # apple
print(fruits[-1])    # cherry
```

### Slicing Tuples

```python
nums = (10, 20, 30, 40, 50)
print(nums[1:4])   # (20, 30, 40)
print(nums[:3])    # (10, 20, 30)
```
