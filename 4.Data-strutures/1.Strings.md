## âœ‚ï¸ String Indexing & Slicing

Strings behave like sequences â†’ each character has an **index**.

```python

print(word[0])    # P
print(word[-1])   # n (last character)

print(word[:3])   # Pyt
print(word[2:])   # thon
```

## ğŸ”§ Common String Methods
