## ğŸ“Œ 5. Using Templates

1. Create a folder named `templates` in your project.
2. Create `index.html` inside `templates`:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Flask App</title>
</head>
<body>
    <h1>Hello, {{ name }}!</h1>
</body>
</html>
```

3. Render template in Flask:

```python

def hello(name):
    return render_template("index.html", name=name)
```

âœ… `{{ name }}` â†’ Jinja2 template variable

## ğŸ¯ Mini Project: Simple Greeting App

```python

@app.route("/")

    return render_template("form.html")

    username = request.form["username"]
    return render_template("greet.html", name=username)

    app.run(debug=True)
```


```html

```
