## ğŸ“Œ 4. Enclosing (Nonlocal) Variables

In **nested functions**, variables from the outer function can be modified using `nonlocal`.

```python

    x = "outer"
    
    def inner():
        nonlocal x
        x = "modified in inner"
    
    inner()
    print(x)

outer()   # modified in inner
```
