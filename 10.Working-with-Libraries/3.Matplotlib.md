## ðŸ“Œ 3. Line Plot

```python
x = [1, 2, 3, 4, 5]
y = [10, 20, 15, 25, 30]

plt.plot(x, y, color='blue', marker='o', linestyle='--', label='Sales')
plt.title("Sales Trend")
plt.xlabel("Day")
plt.ylabel("Sales")
plt.legend()
plt.show()
```

âœ… Shows how values change over time.
