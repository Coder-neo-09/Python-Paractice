## ðŸ“Œ 3. Creating Arrays

```python
import numpy as np

# 1D array
arr1 = np.array([1, 2, 3, 4])
print(arr1)  # [1 2 3 4]

```

## ðŸ“Œ 4. Array Operations

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print(a + b)  # [5 7 9]
print(a * b)  # [4 10 18]
print(a ** 2) # [1 4 9]
```

âœ… Operations are **element-wise** and fast.

---

## ðŸ“Œ 5. Array Functions

```python
arr = np.array([1, 2, 3, 4, 5])

print(np.mean(arr))  # 3.0
print(np.sum(arr))   # 15
print(np.max(arr))   # 5
print(np.min(arr))   # 1
```

---

## ðŸ“Œ 6. Reshape & Indexing

```python
arr = np.arange(1, 10)       # [1 2 3 4 5 6 7 8 9]
arr2d = arr.reshape(3, 3)    # 3x3 array

print(arr2d[0, 1])  # 2 (row 0, col 1)
print(arr2d[:, 0])  # [1 4 7] (all rows, col 0)
```
