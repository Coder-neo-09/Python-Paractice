## ðŸ“Œ 3. Common Assertions

| Assertion                    | Description            |
| ---------------------------- | ---------------------- |
| `assertEqual(a, b)`          | a == b                 |
| `assertNotEqual(a, b)`       | a != b                 |
| `assertTrue(x)`              | x is True              |
| `assertFalse(x)`             | x is False             |
| `assertIsNone(x)`            | x is None              |
| `assertIsInstance(obj, cls)` | obj is instance of cls |
