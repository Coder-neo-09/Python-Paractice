### ‚úñÔ∏è `multiply()` ‚Äì Multiplication

### ‚ûó `divide()` ‚Äì Division (with error handling)

```python
def divide(a, b):
    if b == 0:
        return "Error: Cannot divide by zero"
    return a / b
```

‚úÖ Prevents program crash when dividing by zero.

## ‚ö° 2. Menu-Driven Calculator

```python
def calculator():
    print("üßÆ Simple Calculator")
    print("Choose operation: +  -  *  /")

    choice = input("Enter operator: ")
    a = float(input("Enter first number: "))
    b = float(input("Enter second number: "))

    if choice == "+":
        print("Result:", add(a, b))
    elif choice == "-":
        print("Result:", subtract(a, b))
    elif choice == "*":
        print("Result:", multiply(a, b))
    elif choice == "/":
        print("Result:", divide(a, b))
    else:
        print("Invalid operator")
```

## üß™ 4. Real-World Usage Example

```python
import math

def calculator():
    while True:
        print("\nüìå Menu: +  -  *  /  ^  sqrt  exit")
        choice = input("Enter operator: ")

        if choice == "exit":
            print("üëã Exiting Calculator")
            break

        if choice == "sqrt":
            num = float(input("Enter number: "))
            print("Result:", math.sqrt(num))
        else:
            a = float(input("Enter first number: "))
            b = float(input("Enter second number: "))

            if choice == "+":
                print("Result:", add(a, b))
            elif choice == "-":
                print("Result:", subtract(a, b))
            elif choice == "*":
                print("Result:", multiply(a, b))
            elif choice == "/":
                print("Result:", divide(a, b))
            elif choice == "^":
                print("Result:", power(a, b))
            else:
                print("Invalid operator")
```

‚úÖ Now supports **continuous calculations**, **square root**, and **exit option**.

## üß† Best Practices

* Keep **functions small & reusable** (`add`, `subtract`, etc.)
* Always add **error handling** for invalid inputs (e.g., `divide by zero`)
* Use **loops** to allow multiple calculations without restarting program
* Import **`math` module** for advanced operations
* Extend project gradually (scientific calculator, GUI with Tkinter, etc.)
