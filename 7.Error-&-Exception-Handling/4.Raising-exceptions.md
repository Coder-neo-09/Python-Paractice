## âœ… Example 2: Raising Custom Exception

```python

    if b == 0:
        raise ZeroDivisionError("Cannot divide by zero!")

# print(divide(5, 0)) # âŒ ZeroDivisionError
```

## âœ… Example 3: Using `raise` inside Try/Except

```python

    num = int(input("Enter positive number: "))
    if num < 0:
        raise ValueError("Number must be positive")
except ValueError as e:
    print("Error:", e)
```
