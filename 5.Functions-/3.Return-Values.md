# ðŸ”„ 5.3 Return Values

Functions can **send back data** to the caller using the `return` statement.
This allows us to **store results** and use them later, instead of only printing them.

---

## ðŸ“ Syntax

```python
def function_name(parameters):
    # code
    return value
```

âœ… Example:

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)   # 8
```

---

## ðŸ“Œ Returning Multiple Values

Python can return **multiple values** (as a tuple).

```python
def calculate(a, b):
    return a + b, a - b, a * b

add, sub, mul = calculate(10, 5)
print(add, sub, mul)   # 15 5 50
```

---

## âš¡ Return vs Print

```python
def square1(x):
    print(x**2)   # only displays result

def square2(x):
    return x**2  # sends value back

val = square2(4)
print(val + 10)  # 26
```

âœ… Use `return` if you want to **use the result later**.

---

## ðŸ”„ Return Without Value

If no `return` is given â†’ function returns `None`.

```python
def hello():
    print("Hello")

result = hello()
print(result)   # None
```

## ðŸ§ª Example: Max of Three Numbers
